cgn.regop.regularization_operator module

class RegularizationOperator

Bases: object

Abstract base class for regularization operators. Each child of RegularizationOperator must implement the methods fwd and adj, which give the forward and the adjoint action of the regularization operator.

adj(w)

Evaluates the adjoint action of the regularization operator. :param w: Of shape (r, m), where r = rdim. :return: Of shape (n, m), where n = dim.
